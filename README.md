# ArduinoConnect
Swift Application used to test the digital Pins of your ESP8266 compatible Arduino Board. Use in conjuction with ArduinoConnect.ino Sketch.

## How To Use
째Compile and install the xCode project on your iOS device
째Compile and flash your ESP8266 board with the Firmware (see below for download)
째Enter your ESP's IP(v4) address in the textfield
째Try out the digital pins
